# Users Manual

## Login
1. Open `/login`.
2. Enter credentials.
3. Click `ログイン`.

## ToDo List
- View list at `/todos`.
- Sort by clicking column headers.
- Change page size using the dropdown.

## Create
1. Click `+ 新規作成`.
2. Fill the form.
3. Confirm and register.

## Edit
1. Click `編集` on a row.
2. Update fields.
3. Click `更新`.

## Delete
- Single delete: click `削除`.
- Bulk delete:
  1. Select rows with checkboxes.
  2. Click `選択した項目を削除`.

## Overdue
- Click `期限切れ` to filter overdue items.
- Return to `一覧` to show all.

## Export
- Click `CSVダウンロード` to download the full list.

## Logout
- Click `ログアウト` in the header.
